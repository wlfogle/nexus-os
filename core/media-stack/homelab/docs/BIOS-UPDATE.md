# BIOS Update Information for Tiamat's Gigabyte B450M DS3H WIFI-CF Motherboard

**Current BIOS Version:** [Insert current version]

## Update Instructions:

1. **Download the BIOS Update**: Visit the official Gigabyte website and navigate to the support section for the B450M DS3H WIFI-CF motherboard. 
2. **Prepare the USB Drive**: Format a USB drive to FAT32 and copy the downloaded BIOS update file onto it.
3. **Enter BIOS Setup**: Restart your computer and repeatedly press the `DEL` key during boot to enter the BIOS setup.
4. **Backup Current BIOS**: Before proceeding with the update, it's advised to create a backup of the current BIOS version, if your motherboard supports this feature.
5. **Update the BIOS**:
   - Use the `Q-Flash` utility or the integrated BIOS update feature to select the BIOS file located on the USB drive.
   - Follow the on-screen prompts to complete the update.
6. **Reboot**: Once the update is finished, reboot your system to apply the changes.

## Important Notes:
- Ensure that your system is connected to a reliable power source during the update process.
- Do not turn off your computer or reset the BIOS during the update process to prevent corruption.
